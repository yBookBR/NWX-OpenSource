# NWX Open Hub

The NWX Open Hub is a project entirely developed by Book. 
The script is written in **Lua/Luau** and is completely **open source** (see the license for more information).
This project includes the main interface and several features of the original **NWX-Hub** script, also developed by Book.

### features

- Modular and organized user interface
- Server information (Players, Ping, Server ID)
- ESP
- Universal scripts
- Speed ​​slider with dynamic adjustment

### about the Project

The goal of the NWX Open Hub is to provide a structured and modular base, facilitating modification, expansion, and use for learning Luau/Lua.
### How to Use
Use an executor such as KRNL, XENO, Fluxus, Delta, or others. Copy the load string and execute it in your Roblox client.


### Loadsting:
```lua
local url = "https://pastebin.com/Z10q8KD8"
loadstring(game:HttpGet(url))()
```
